manage module¶

Funzioni utili per la gestione del server da riga di comando

manage.addDocument(url)[sorgente]¶

Aggiungi un documento a fuseki

Parametri:string – url L’URL del documento da aggiungere
manage.apacheConf()[sorgente]¶

Genera la configurazione di Apache necessaria per installare Raschietto.

Avvertimento

Probabilmente è un’idea migliore eseguire l’installazione a manoni...

manage.build()[sorgente]¶

Compila tutti i file statici di Raschietto

manage.gui()[sorgente]¶

Avvia la ‘splendida’ GUI di RaschiettoManager

manage.info()[sorgente]¶

Stampa una lista di variabili d’ambiente relative a Flask nella forma chiave==valore

manage.setup(target='all')[sorgente]¶

Installa tutte le dependencies di Raschietto e compilaa i file statici

manage.watch()[sorgente]¶

Compila tutti i file statici di Raschietto e rimane in attesa di ulteriori modifiche da ricompilare
